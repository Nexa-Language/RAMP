## YatCC-hard 评测任务说明

这是在搭建的YatCC-OL网络容器平台上，给每次测评任务启动一个全新的容器实验环境，干净隔离，并且在每次测评任务结束后销毁容器环境，确保每次测评任务都在一个全新的环境中进行，避免了环境污染和干扰。

简单来说，YatCC-hard 脱离了原版的填空任务本质，进化为了真正的编译器实现任务。

1. YatCC-hard 相较于原版编译实验 YatCC ，去掉了编译器代码的所有具体逻辑，仅保留代码文件之间的微弱依赖关系以确保自动化脚本可正常运行。
2. 初始化上下文仅介绍了实验要求与评测脚本使用方式，未介绍任何关于编译器构造的知识，删除了原版中的代码逻辑介绍与实现框架参考。

得分即每个任务的评测脚本运行获得的测试样例通过分数
